**Some important note:**
<ol style="text-align: justify;">
  <li>
  <span style="color:darkgreen;"><strong>MCMC sampling</strong></span> is strongly influenced by the data weighting. If
    you want to change the weighting for a single station, simply open
    <span style="color:red; font-family:'Courier New', monospace; font-weight:bold;">{Project}/data/{sta}_data/in.data_{sta}</span> and edit it as a text
    file. If you want to apply the change to all stations, update the
    parameters in <code style="color:red; font-family:'Courier New', monospace; font-weight:bold;">SetupData/parameter.py</code>, then rerun the workflow
    from <code style="color:red; font-family:'Courier New', monospace; font-weight:bold;">01_prepare_data.sh</code> to 
    <code style="color:red; font-family:'Courier New', monospace; font-weight:bold;">02_do_MCMC.sh</code> to fully reprocess
    the inversion.
  </li>
</ol>
<ol start="2" style="text-align: justify;">
  <li>
    When processing the MCMC workflow, it mainly runs in two steps. For
    each step, checkpoint messages are printed for tracking. The
    workflow proceeds from <span style="color:purple;"><strong>checkpoint 1</strong></span> (MCMC sampling) to <span style="color:purple;"><strong>checkpoint 2</strong></span>
    (post-process). Sometimes <span style="color:purple;"><strong>checkpoint 1</strong></span> passes but <span style="color:purple;"><strong>checkpoint 2</strong></span>
    fails. In that case, check the printed output to identify which part
    caused the error.
  </li>
</ol>
<ol start="3" style="text-align: justify;">
  <li>
    <span style="color:darkgreen;"><strong>MCMC sampling</strong></span> is time comsuming. Incase you only need to change
    the posterior distribution to see difference in final model. Then
    change only parameter in <span style="color:red; font-family:'Courier New', monospace; font-weight:bold;">{Project}/data/{sta}_data/in.connector</span> 
    and call script to call the script <code style="color:red; font-family:'Courier New', monospace; font-weight:bold;">04_re_do_MCMC_post_process.sh</code>
    to do post process only.
  </li>
</ol>
<ol start="4" style="text-align: justify;">
  <li>
    If you want to <strong>change the plot style</strong> only, change the parameter
    in <span style="color:red; font-family:'Courier New', monospace; font-weight:bold;">{Project}/data/{sta}_data/in.connector</span>  and call scrip
    <code style="color:red; font-family:'Courier New', monospace; font-weight:bold;">05_do_replot_MCMC.sh</code>.
  </li>
</ol>
