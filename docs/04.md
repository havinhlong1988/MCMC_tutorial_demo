**3. How the MCMC works**

The code mainly runs in two main steps + 1 plot step:

<strong>(1) Run </strong><span style="color:darkgreen;"><strong>MCMC sampling</strong></span>: The code performs <span style="color:darkgreen;"><strong>MCMC sampling</strong></span>
 by drawing model parameters, generating posterior samples, and calculating
the misfit between synthetic and observed data. Models that satisfy the
acceptance condition are stored in binary files.
This step is executed by <code style="color:red; font-family:'Courier New', monospace; font-weight:bold;">sh 02_do_MCMC.sh</code>, which calls the sub-script
<code style="color:red; font-family:'Courier New', monospace; font-weight:bold;">{Project}/MonterCarlo/01.run_MCMC.sh</code>.

<strong>(2)</strong> </strong><span style="color:darkgreen;"><strong>Post-process posterior models</strong></span>: The code reads the binary
files from step <strong>(1)</strong> and selects a subset of models based on the settings
in <span style="color:red; font-family:'Courier New', monospace; font-weight:bold;">{Project}/data/{sta}_data/in.connector</span>. It then calculates the
average model, standard deviation, and performs forward modeling to
generate synthetic data for plotting.
This step is executed by <code style="color:red; font-family:'Courier New', monospace; font-weight:bold;">sh 02_do_MCMC.sh</code>, which calls
<code style="color:red; font-family:'Courier New', monospace; font-weight:bold;">{Project}/MonterCarlo/02.run_MCMC_post_process.sh</code>.

<strong>(3)</strong> </strong><span style="color:darkgreen;"><strong>Plot results</strong></span>: The final figures are generated using the
selected models and corresponding synthetic data from step <strong>(2)</strong>. This
step is excuted by <code style="color:red; font-family:'Courier New', monospace; font-weight:bold;">AnalyzeResult/inversion_plot_vfinal_flex.py</code>.Output figures are saved in <span style="color:red; font-family:'Courier New', monospace; font-weight:bold;">{Project}/MonterCarlo/{sta}/*.png</span>.
